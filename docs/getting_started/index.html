    Getting started
      • Dependencies
      • Example:

    Add any of these lines to your build.sbt:

    // On JVM
libraryDependencies += "io.github.pityka" %% "nspl-awt" % "0.1.0+6-488a1f6c-SNAPSHOT"
// In browser, using the canvas renderer
libraryDependencies += "io.github.pityka" %%% "nspl-canvas-js" % "0.1.0+6-488a1f6c-SNAPSHOT"
//

    Dependencies #

    • nspl-awt depends on vectorgraphics2d
    • nspl-canvas-js has no external dependencies besides the scalajs-dom from the Scala Js project.
